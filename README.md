# PowerShell Remote Support Framework

A reusable PowerShell framework for safe IT support scripts.

## Features

- Admin detection
- Timestamped report folders
- Logging helper
- Confirmation helper
- Export helper
- Menu pattern
- Sample diagnostic command

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\PowerShell_Remote_Support_Framework.ps1
```

## Safety

Framework only. It provides safe structure for future support tools.
